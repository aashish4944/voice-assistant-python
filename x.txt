# Voice Assistant using Python

## Setup Instructions (Windows)

1. Install Python 3.11

2. Install required libraries:
   pip install pyttsx3 speechrecognition pyjokes pyaudio

3. Run the project:
   python assistant.py

## Features

* Voice recognition
* Text-to-speech output
* Tell time
* Open Google
* Generate jokes
* Set alarm using voice
